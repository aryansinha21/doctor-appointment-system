# Doctor Appointment Client

A simple React frontend for the Doctor Appointment System.

## Run locally

1. Open a terminal in `client`
2. Run `npm install`
3. Run `npm run dev`

The app will be available at `http://localhost:5173` and uses the Vite `/api` proxy by default (pointing to `http://localhost:5000`).

For production or remote backends, set `VITE_API_URL` (for example in `.env`):

```env
VITE_API_URL=https://your-backend-domain.com
```
